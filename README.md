## Pendahuluan

Dalam pengembangan aplikasi perbankan berbasis web, ada beberapa elemen HTML, CSS, dan JavaScript yang sering digunakan. Berikut adalah panduan dari A hingga Z mengenai elemen-elemen tersebut.

### HTML (HyperText Markup Language)

#### A - `<a>`
Tag `<a>` dalam HTML digunakan untuk membuat hyperlink, yang memungkinkan pengguna untuk mengklik dan diarahkan ke halaman atau sumber lain. Berikut adalah contoh sederhana dan cara penggunaannya:

### Contoh Dasar

```html
<a href="https://www.example.com">Kunjungi Situs</a>
```

Dalam contoh ini:
- `href="https://www.example.com"` adalah atribut yang menentukan URL tujuan.
- Teks "Kunjungi Situs" adalah yang akan ditampilkan dan dapat diklik oleh pengguna.

### Contoh dengan Atribut Tambahan

```html
<a href="https://www.example.com" target="_blank" rel="noopener noreferrer">Kunjungi Situs</a>
```

Penjelasan:
- `target="_blank"`: Membuka tautan di tab atau jendela baru.
- `rel="noopener noreferrer"`: Menghindari potensi risiko keamanan saat membuka tautan di tab baru, dan menghilangkan referer dari tautan.

Jadi, `<a>` adalah elemen yang sangat penting dalam HTML untuk navigasi antara halaman atau dokumen.

#### B - `<button>`
Elemen tombol yang digunakan dalam formulir atau untuk interaksi pengguna.

#### C - `<code>`
Digunakan untuk menampilkan kode dalam halaman web.

#### D - `<div>`
Elemen kontainer yang digunakan untuk membungkus elemen lain.

#### E - `<input>`
Elemen input dalam formulir, seperti teks, radio, atau checkbox.

#### F - `<form>`
Elemen formulir yang digunakan untuk mengumpulkan data dari pengguna.

#### G - `<h1> - <h6>`
Elemen judul dengan tingkat kepentingan yang berbeda.

#### H - `<header>`
Bagian atas dari sebuah halaman atau bagian halaman.

#### I - `<img>`
Elemen gambar yang digunakan untuk menampilkan gambar di halaman web.

#### J - `<label>`
Elemen label untuk elemen input dalam formulir.

#### K - `<kbd>`
Elemen yang digunakan untuk menampilkan input keyboard.

#### L - `<link>`
Digunakan untuk menghubungkan halaman ke file CSS eksternal.

#### M - `<meta>`
Elemen metadata untuk mendefinisikan informasi tentang halaman.

#### N - `<nav>`
Bagian navigasi pada halaman web.

#### O - `<ol>`
Elemen daftar yang diurutkan.

#### P - `<p>`
Elemen paragraf teks.

#### Q - `<q>`
Elemen kutipan singkat.

#### R - `<script>`
Elemen yang digunakan untuk menyertakan kode JavaScript.

#### S - `<section>`
Bagian terpisah dari halaman web, biasanya dengan heading.

#### T - `<table>`
Elemen tabel untuk menyajikan data dalam format tabel.

#### U - `<ul>`
Elemen daftar yang tidak diurutkan.

#### V - `<video>`
Elemen video untuk menyematkan video dalam halaman web.

#### W - `<form>`
Elemen formulir yang digunakan untuk mengumpulkan data dari pengguna.

#### X - `<textarea>`
Elemen input area teks yang lebih besar daripada `<input>`.

#### Y - `<a>`
Elemen tautan yang digunakan untuk membuat hyperlink.

#### Z - `<footer>`
Bagian bawah dari halaman atau bagian halaman.

### CSS (Cascading Style Sheets)

#### A - `align-items`
Properti Flexbox untuk menyelaraskan item di sepanjang sumbu silang.

#### B - `background-color`
Properti untuk mengatur warna latar belakang elemen.

#### C - `color`
Properti untuk mengatur warna teks.

#### D - `display`
Properti yang mengatur jenis tampilan elemen, seperti `block`, `inline`, `flex`, dan `grid`.

#### E - `edge`
Bagian dari box model yang dapat diatur dengan `margin`, `border`, `padding`.

#### F - `flex`
Properti Flexbox untuk menentukan bagaimana item fleksibel dalam container flex.

#### G - `grid`
Properti Grid Layout untuk mengatur tata letak berbasis grid.

#### H - `height`
Properti untuk mengatur tinggi elemen.

#### I - `index`
Properti `z-index` untuk menentukan lapisan elemen dalam konteks stacking.

#### J - `justify-content`
Properti Flexbox untuk mengatur perataan item di sepanjang sumbu utama.

#### K - `keyframes`
Aturan untuk mendefinisikan animasi CSS.

#### L - `layout`
Properti yang mengatur tata letak elemen, seperti `flex` dan `grid`.

#### M - `margin`
Properti untuk mengatur ruang di luar batas elemen.

#### N - `none`
Nilai untuk menghapus properti seperti `display: none`.

#### O - `opacity`
Properti untuk mengatur transparansi elemen.

#### P - `padding`
Properti untuk mengatur ruang di dalam batas elemen.

#### Q - `quotes`
Properti untuk mendefinisikan kutipan dalam elemen `q` dan `blockquote`.

#### R - `resize`
Properti untuk menentukan apakah elemen dapat diubah ukurannya oleh pengguna.

#### S - `shadow`
Properti untuk menambahkan bayangan pada elemen, seperti `box-shadow` dan `text-shadow`.

#### T - `transform`
Properti untuk menerapkan transformasi seperti rotasi, skala, dan penerjemahan.

#### U - `underline`
Properti untuk memberikan garis bawah pada teks.

#### V - `visibility`
Properti untuk mengatur visibilitas elemen, seperti `visible` dan `hidden`.

#### W - `width`
Properti untuk mengatur lebar elemen.

#### X - `x-axis`
Biasanya terkait dengan properti seperti `transform` untuk penerjemahan sumbu X.

#### Y - `y-axis`
Biasanya terkait dengan properti seperti `transform` untuk penerjemahan sumbu Y.

#### Z - `z-index`
Properti untuk menentukan lapisan stacking elemen.

### JavaScript

#### A - `addEventListener`
Metode untuk menambahkan event listener ke elemen.

#### B - `Boolean`
Tipe data untuk nilai true atau false.

#### C - `console.log`
Metode untuk menampilkan pesan di konsol.

#### D - `DOM`
Document Object Model, representasi struktur dokumen HTML.

#### E - `event`
Objek yang menyimpan informasi tentang peristiwa yang terjadi.

#### F - `for`
Looping statement untuk iterasi melalui koleksi data.

#### G - `getElementById`
Metode untuk mendapatkan elemen berdasarkan ID.

#### H - `hash`
Bagian dari URL setelah tanda #, sering digunakan dalam routing.

#### I - `if`
Pernyataan kondisional untuk eksekusi kode berdasarkan kondisi.

#### J - `JSON`
Format data untuk pertukaran data antara server dan client.

#### K - `key`
Properti objek atau nilai dalam koleksi data.

#### L - `localStorage`
API untuk menyimpan data secara lokal di browser.

#### M - `map`
Metode array untuk menerapkan fungsi ke setiap elemen.

#### N - `null`
Nilai yang menunjukkan bahwa variabel tidak memiliki nilai.

#### O - `object`
Tipe data untuk menyimpan kumpulan nilai dengan kunci.

#### P - `Promise`
Objek untuk menangani operasi asynchronous.

#### Q - `querySelector`
Metode untuk memilih elemen menggunakan selector CSS.

#### R - `return`
Perintah untuk mengembalikan nilai dari fungsi.

#### S - `setTimeout`
Metode untuk menjalankan fungsi setelah delay tertentu.

#### T - `this`
Kata kunci yang merujuk pada konteks eksekusi saat ini.

#### U - `undefined`
Nilai yang menunjukkan bahwa variabel belum didefinisikan.

#### V - `variable`
Tempat untuk menyimpan data dalam program.

#### W - `while`
Looping statement yang menjalankan kode selama kondisi benar.

#### X - `XMLHttpRequest`
Objek untuk melakukan permintaan HTTP.

#### Y - `yield`
Kata kunci dalam generator functions untuk mengembalikan nilai satu per satu.

#### Z - `zero`
Angka 0, sering digunakan dalam perhitungan dan perbandingan.
